## Lien Heroku du projet

https://i-love-forms-too.herokuapp.com/

**Si vous voulez télécharger le repo, lancez**

```bundle install```  
```rails db:create ```

```rails db:migrate```   

Accès à la console pour vérifier la structure de la BDD
```rails console```
